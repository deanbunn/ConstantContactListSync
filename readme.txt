Constant Contact List Sync

A quick C# console project to demostrate how to pull Constant Contact lists and add email addresses to a list. 

Added email addresses will be created as contacts if they don't already exist in the client's account. 

API requires a Constant Contact developer API key and uses OAuth 2.0. They don't expire the token.

To get developer API key and OAuth token follow instructions listed here:
http://developer.constantcontact.com/home/api-keys.html
